# NIVAA Admin Panel

A premium management interface for the NIVAA e-commerce platform.

## Features

- **Dashboard**: Overview of sales, orders, and customers.
- **Product Management**: List, search, and filter products to manage inventory.
- **Modern UI**: Built with React, TypeScript, and Tailwind CSS.
- **Premium Aesthetics**: Follows the NIVAA brand guide with the "Atelier Stone" palette.

## Getting Started

1. **Install Dependencies**:

   ```bash
   npm install
   ```

2. **Run in Development**:

   ```bash
   npm run dev
   ```

3. **Build for Production**:
   ```bash
   npm run build
   ```

## Tech Stack

- **React 18+**
- **TypeScript**
- **Tailwind CSS**
- **Lucide React** (Icons)
- **Framer Motion** (Animations)
- **Vite** (Build Tool)
